# Recipe Step Images

This directory contains images for recipe instruction steps.

## Image Naming Convention

Images follow this pattern: `{recipe-slug}-step{number}[-{index}].jpg`

Examples:
- `moussaka-step1.jpg` - Single image for step 1
- `moussaka-step3-1.jpg`, `moussaka-step3-2.jpg` - Two images for step 3
